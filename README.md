# Practica-7-Glupix_Farm
GLUPIX FARM ESCAPE es un juego estilo voxel inspirado en Crossy Road, donde controlas a Glupix, un alien verde que aterrizó en una granja con la intención de llevarse unas vacas, pero terminó metido en problemas. Ahora debe escapar avanzando por un camino infinito lleno de obstáculos, animales y peligros.
